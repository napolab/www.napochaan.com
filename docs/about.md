# なぽりたん（naporitan / napochaan / naporin）

> おそろしき、なんでも屋。さまざまな「破壊」承っております。
> フルスタックエンジニア / DJ / VJ。リアルと VR の境界を歪ませる。

ぼくは東京を拠点に活動するプログラマーで、同時に DJ / VJ でもあります。
専門はウェブサイト制作・Web アプリケーション・モバイルアプリの開発と運営。
多才なクリエイターが集う [StudioGnu](https://studiognu.org) に所属し、様々なプロジェクトで技術面の貢献をしています。
コードと音と映像、どれも「素材を分解して、別のかたちに組み直す」という同じ衝動から作っていて、
技術的な興味は一貫して **dissolution（崩壊）・glitch・fragmentation（断片化）** に向いています。

## Creative Philosophy

ぼくの作るものには、いつも **「壊す → ほどける → 組み直す」** という流れがあります。
ピクセルが溶け、画像が破断し、ビートに合わせて像が崩れていく——その不安定さの中の一瞬を掴まえたい。
そしてその崩壊は、誰かに見せる・踊らせる・没入させるための装置でもある。
ソロの作品として完結させるより、**人の表現が立ち上がる場をつくる**方に、ぼくは創作の手応えを感じています。
